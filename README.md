# Building Li Xun's Heart Code (But Better!) 💖

Hey! Thanks for stopping by. 

If you watched the C-drama *Lighter and Princess*, you already know exactly why I made this. Li Xun's legendary particle heart code completely stole the show, and like any certified fan, I just *had* to see if I could build it myself. 

Instead of just copy-pasting the standard fan code floating around the internet, I wanted to give it a major glow-up. I tinkered with the math and added a few features to make it feel a lot more alive and responsive.

## 🥰 Why this version hits different

* **That Perfect Tone:** I ditched the bright, blinding neon pinks. Instead, I locked down a super rich, moody crimson red. It looks incredibly clean and sharp against the pitch-black window.
* **Hidden Messages:** If you click anywhere on the screen, a burst of mini-particles explodes, and a secret phrase gently drifts up into the dark before fading away. 
* **The Magic Wand Effect:** Moving your mouse pointer across the screen leaves a trailing wave of glowing red dust behind it. It feels like you're literally painting with embers.

## 🤫 Make it your own secret message board
You don't need to dig through hundreds of lines of complex math to change the text. I set up a dedicated playground right at the top of the `heart.py` file around line 18:

```python
TEXT_POOL = [
    "I Love You", 
    "Forever 💖"
]
```
You can change those words to inside jokes, names, or whatever phrases you want to show off when someone clicks the screen!

## 🚀 Play with it right now
1. Make sure you have **Python** installed on your system.
2. Download my `heart.py` file.
3. Right-click the file and hit **Edit with IDLE**.
4. When the code window pops up, press **F5** on your keyboard.
5. Move your mouse around and start clicking to see the magic happen!
